## Component Placement

### Fixed/Sticky Components
- **TopNavbar**: Fixed at top (z-50)
- **LeftSidebar**: Sticky on desktop, fixed overlay on mobile (z-50)
- **ChatBot**: Fixed bottom-right (z-50)

### Main Content Flow (Top to Bottom)
1. **HeroBanner** - Full width hero section
2. **Greeting Section** - "Hey Vaishno, Happy To See You!"
3. **Quick Access Row** (2 columns)
   - TimeWidget (left)
   - HolidayCalendar (right)
4. **QuickLinks** - 3 column grid
5. **MoodTracker** - 4 emoji buttons
6. **OrganizationEngagement** - Post/Poll widget
7. **UpcomingApps** - 5 app icons
8. **Footer** - Full width footer

### Right Sidebar (Desktop Only)
- **RecommendedSection** - Scrollable content cards
